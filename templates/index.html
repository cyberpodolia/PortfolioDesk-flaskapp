Local Planner
Click to enlarge

Local-first daily planner for Windows. Runs fully on localhost (127.0.0.1) with FastAPI + SQLite and a built-in scheduler. Optimized for daily execution, KPI visibility, and AI-assisted planning via a pluggable LLM layer - local models first, with cloud LLMs optional.

Key features

  • Local-first / offline-capable: localhost-only backend + SQLite as source of truth; core works without cloud
  • Pluggable LLM automation: supports local LLMs (preferred) or optional cloud LLMs for planning, rewrites, and summaries
  • KPI dashboard: last N days (done/cancelled/failed/overdue) + minutes + score
  • Desktop widget: quick view + fast actions without opening the full UI
  • Daily task tracking: Planned -> Active -> Done / Overdue, quick notes + full task detail pages
  • Kanban board: lightweight board view for current workflow
  • Day plan rewrite: AI/CLI can regenerate today's plan as a clean snapshot (no append bloat)
  • Agent-friendly API/CLI: deterministic endpoints; acknowledgements + comments stored and visible in UI/API
  • Knowledge Base linking: tasks reference KB notes; exports plans/reports/artifacts into the vault
  • Telegram alerts: concise summaries and reminders (rate-limited)

Project status

Stage 1 - Personal Prototype (business-plan validation): proving the workflow and metrics in a real daily loop before packaging/distribution.
